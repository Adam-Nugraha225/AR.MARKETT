Project information

  • Category: Programming
  • Client: Katrina Owen
  • Project date: 2013
  • Project URL: www.exercism.com

Exercism.org is a programming learning platform that provides free coding challenges and exercises for a variety of programming languages. Users can choose from over 50 programming languages, including Python, JavaScript, Ruby, and many others. The platform offers a unique approach to learning, where users complete coding exercises and then get feedback from experienced volunteer mentors.
